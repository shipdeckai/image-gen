Get Started in Seconds

Install as a Claude Code plugin or use standalone.

# Install as Claude Code plugin
claude plugin add shipdeckai/image-gen

# Or clone and use directly
git clone https://github.com/shipdeckai/image-gen
cd image-gen && npm install

# Generate an image
node dist/cli.js generate "A golden sunset over mountains"
