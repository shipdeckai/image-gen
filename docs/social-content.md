## Reddit Post (r/ClaudeAI)

**Title:** I built a Claude Code plugin that unifies 10 AI image generation providers [Open Source]

**Body:**

I got tired of switching between different image generation APIs depending on what I needed. DALL-E for text, FLUX for photorealism, Ideogram for logos... it was a mess.

So I built **Image Gen** - a Claude Code plugin that unifies all 10 providers into one interface.

**What it does:**

- **Intelligent selection** - Claude automatically picks the best provider for your prompt (text-heavy? Ideogram. Photorealistic? BFL FLUX)
- **Automatic fallbacks** - If one provider fails, it tries the next. Your workflow never breaks
- **10 providers** - OpenAI DALL-E 3, BFL FLUX.2, Stability AI, Ideogram, Google Gemini, FAL, Leonardo, Recraft, Replicate, ClipDrop

**Quick install:**

```
/plugin marketplace add shipdeckai/claude-skills
/plugin install image-gen@shipdeckai/claude-skills
```

Then just ask Claude:

> "Generate a professional product photo of a coffee mug"

It picks the right provider automatically.

**Why 10 providers matters:**

| Provider | Best For |
|----------|----------|
| OpenAI DALL-E 3 | General purpose, text rendering |
| BFL FLUX.2 | Photorealism, 4K resolution |
| Ideogram v3 | Typography, logos |
| Recraft v3 | #1 ELO ranked, vectors |
| Leonardo | Artistic, fantasy, characters |

Full list + docs: https://shipdeckai.github.io/image-gen

It's MIT licensed and free to use. Would love feedback from the community.
